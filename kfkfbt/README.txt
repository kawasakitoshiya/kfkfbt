(0)NXTとPCの接続方法
NXTのブルートゥースをONにし、標準ファームウェアの画面で待機
PCのbluetoothをONにする。
windowsメニュー
-> デバイスとプリンター
->デバイスの追加
表示されている中でそれっぽいやつを選択。
NXT側でパスコードが表示されるのでNXTで決定する。
PC側でNXTに表示されていたパスキーをPC側で入力
ペアリング完了。
(1)COMって？
windowsメニュー
-> デバイスとプリンター
-> 接続されたNXT機のアイコンで右クリックでプロパティ
-> [ハードウェア］タブの名前に書いてある[COM*]のこと
->もしくは[サービス]タブに書いてあるかもしれないです。
